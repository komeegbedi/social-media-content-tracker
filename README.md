# IFC Social Media Content Planner

A collaborative content planning and tracking app for the IFC Social Media team.

This project is being built to replace our current Google Sheets workflow with a dedicated web application that improves visibility, ownership, and collaboration when planning and publishing content.

---

## MVP Reference

The current MVP is a Google Sheet that the team actively uses to plan and track social media content.

A **screenshot of this MVP Google Sheet** is included in this repository and serves as the source of truth for:
- Data structure
- Columns and statuses
- Workflow and process

All features in this app are modeled directly from that sheet.

---

## Goal

Recreate the existing workflow in a React-based app and gradually enhance it with better UX, review flows, and collaboration tools.
